Problem B

问题等价于：m个+1和n个-1组成数列（m > n），求部分和为正数的概率。
这是个卡特兰数变形问题。
所有可能数列共C(n, m+n)种情况。
现考虑不符合条件的情况。对于不符合条件的数列，设其最早在第i位部分和为-1，对0~i位取反得到映射数列，这个数列有m+1个+1和n-1个-1.
而对于任意一个有m+1个+1和n-1个-1的数列，设其最早在第j位部分和为+1，把0~j取反得到的数列为不符合条件数列）。
由上可知，不符合条件的数列组成的集合和映射数列组成的集合一一对应。
而映射数列的个数易得：C(n-1, m+n)
则所求的概率为：(C(n, m+n) - C(n-1, m+n)) / C(n, m+n)

Problem C

题目给出L个左括号和R个右括号，问构造的排列最多能得到多少个合法的非空子括号串
最优情况是：()()()…()，其中小括号个数是min(L,R)
证明：

  1. 构造的串中去掉孤立的单边括号，所得到的结果数目不变。
  2. 构造的串中若存在包含结构(S)，则结果为S的结果加一，而把串重构成S()则可得到更大的结果，故串中不应存在包含结构
    满足以上条件的串只有:()()()…()
    数目为 (1+min(L,R)) * min(L,R) / 2
    注意：不能直接用公式计算，乘法会溢出。而应循环迭代。
